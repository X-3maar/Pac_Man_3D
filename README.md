# Pac_Man_3D
Pac Man game 3d version! if you never heard about pacman ist is a 2d retro game where you are trying to eat all the pellets in a map with 4 ghosts chasing you non-stop. Since this game is kinda old and has a pretty boring design, I remade a new version of it in 3d with better graphics!

## Controls
Just like the basic movement in the real 2d game, you can move left, right, up and down using the arrows or WASD, and also pause the game using the escape button.
## How to play
After the game starts, you should move and collect all the pellets, but you will find some pellets that are bigger in size than the rest (exactly four) , these pellets can make you move faster, make ghosts feaer you and even have the ability to eat the ghosts for 7 seconds. But be careful, after the 7 seconds finish they will start chasing you again.
After you collect all the pellets in the map, the ghosts will disappeaar and you will win. To restart the game, eat the pellet where the text points.
## Further Details
This game is made with godot game engine, and all the assets (except for the audio) are complete manual design in using CSG shapes (constructive solid geometry) and mesh instances in godot.
